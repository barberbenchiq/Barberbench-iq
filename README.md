# BarberBench IQ™

**Know Your Numbers. Grow Your Business.**

An interactive founder-beta web app for barbers to track weekly business performance and receive simple, actionable coaching.

## Included in this version

- Responsive marketing landing page
- Interactive founder-beta dashboard
- Weekly KPI entry form
- Revenue, hourly rate, average ticket and rebooking calculations
- Business health score
- Coaching opportunity and annual impact calculation
- Browser storage so demo numbers persist between visits
- Vercel-ready Next.js App Router structure

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push these files to the root of the GitHub repository.
2. Import the repository in Vercel.
3. Keep the detected framework as **Next.js**.
4. Click **Deploy**.

No environment variables are required for this founder-beta demo.
